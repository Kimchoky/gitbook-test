# DNS 관리 예제

도메인관리 사이트에서 NameServer를 우선설정해야 함. 이 NameServer에서 DNS 설정이 가능하다.

A Record : IP만 입력가능. 주 도메인의 IP를 입력해야 한다.

CNAME : 별칭으로, DNS 또는 A레코드 명칭을 입력할 수 있다. 마지막에 "."을 붙여준다.

MX : 메일서버, 우선순위 필요.
